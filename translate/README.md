## News

sparkastML's first translation model is now available!

### Details

- **Training Time:** 5 hours, 20,000 steps
- **Training Device:** RTX 3080 (20GB)
- **Corpus Size:** Over 10 million sentences
- **Validation BLEU Score:** 17
